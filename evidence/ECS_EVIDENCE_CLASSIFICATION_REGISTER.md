## The Five Evidence Types

| Type | Code | Definition |
|---|---|---|
| Observed Primary Data | TYPE_A | Direct reading from authenticated primary source — folio transcript, BNE image, physical measurement, published primary edition |
| Reproducible Methods | TYPE_B | Methods or findings that any trained researcher can replicate using the same sources and procedure |
| Interpretive Models | TYPE_C | Analytical interpretation grounded in TYPE_A/B evidence; represents the most probable explanation consistent with the data |
| Speculative Extrapolation | TYPE_D | Working hypothesis beyond what the data directly supports; requires further investigation or independent corroboration |
| Anecdotal / Non-replicated | TYPE_E | Single-source, unverified, or researcher-specific observation that has not been independently confirmed |

### Critical Rule

**No TYPE_C statement may appear as TYPE_A fact.**

Any sentence presenting an interpretive model (TYPE_C) must include qualifying language: *candidate framework, testable model, interpretive hypothesis, provisional reconstruction, most probable explanation consistent with,* or similar.

Any sentence presenting speculative extrapolation (TYPE_D) must include: *working hypothesis, speculative model, requires further investigation,* or similar.

## Firewall Compliance Checklist

Before any document proceeds to peer review, every paragraph in TYPE_C and TYPE_D sections must be verified against this checklist:

- [ ] Does the paragraph contain any unqualified certainty language ('proves', 'definitively', 'confirms beyond doubt')?
- [ ] Does any TYPE_C interpretive claim appear without a qualifying phrase?
- [ ] Does any TYPE_D speculative claim appear without marking as 'working hypothesis' or 'speculative model'?
- [ ] Has any secondary claim been presented as TYPE_A when it is actually TYPE_C?
- [ ] Has any enhanced image been presented as primary evidence?

Any 'YES' answer requires a revision before peer review submission.
