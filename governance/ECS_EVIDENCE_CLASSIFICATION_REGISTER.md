## Classification Schema

| Type | Name | Definition |
|---|---|---|
| TYPE_A | Primary Observed Data | Direct folio citation, confirmed BNE image, authenticated physical measurement |
| TYPE_B | Reproducible Method | Calculation, geometry, or procedure that can be independently replicated |
| TYPE_C | Interpretive Model | Synthesis or argument derived from TYPE_A and TYPE_B; not directly observable |
| TYPE_D | Speculative Extrapolation | Working hypothesis extending beyond confirmed evidence; requires further testing |
| TYPE_E | Anecdotal / Non-replicated | Single-source, unverified, or unconfirmed claim |

### Firewall Rule
**No TYPE_C statement may appear as TYPE_A fact.**
All TYPE_D claims must be explicitly marked as speculative.
TYPE_E material must be isolated from primary argument.
