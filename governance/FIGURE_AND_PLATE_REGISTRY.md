## Governing Policy

Every image used in this corpus must have a documented source repository, shelfmark, processing disclosure, evidential tier, permission status, and caption. Enhanced imagery is always marked PROVISIONAL / REPLICATION REQUIRED. No enhanced image may be treated as primary evidence.

---

## Permission Status Definitions

| Status | Meaning |
|---|---|
| PENDING | Permission request not yet submitted |
| REQUESTED | Formal request submitted, response awaited |
| GRANTED | Written permission received — documentation on file |
| STANDARD_USE | Repository permits scholarly/educational use without individual permission |
| RESTRICTED | Permission denied or use requires purchase |
| OWN_WORK | Original image created by RSV — no external permission required |

## Enhancement Disclosure Policy

All computationally enhanced, filtered, or processed images must be captioned:

> **"PROVISIONAL IMAGE — COMPUTATIONALLY PROCESSED. Source: [repository + shelfmark]. Processing: [description of enhancement]. This image has not been authenticated as primary evidence. Replication required before use as evidential support. Evidence Tier: TYPE_C or TYPE_D."**
