## Governing Principle

All public-facing prose in this corpus must maintain evidential caution, methodological clarity, and falsifiability. Uncontrolled certainty language undermines scholarly credibility and misrepresents the evidential status of claims.

---

## Banned Language

The following terms and constructions are prohibited in all public-facing documents:

| Banned Term / Phrase | Reason | Use Instead |
|---|---|---|
| proof / proven / proves | Implies logical certainty unavailable in historical scholarship | "the evidence is consistent with" / "supports the hypothesis that" |
| definitely / definitive | Overclaims certainty | "strongly suggests" / "is consistent with" |
| Leonardo definitively created the Shroud | TYPE_D claim stated as TYPE_A fact | "the Shroud image is consistent with a machine-mediated transfer process Leonardo demonstrably understood" |
| hidden code | Conspiracy framing — not appropriate for scholarly argument | "encoded structural relationship" / "compositional regularity" |
| cipher | Same as above | "systematic organisation" / "encoded pattern" |
| conspiracy | Pejorative and non-scholarly | Not used |
| institutional validation | Implies endorsement that does not exist | "submitted for specialist review" |
| academic acceptance | Implies peer review that has not occurred | "consistent with published scholarship" / "not contradicted by published literature" |
| revolutionary discovery | Sensationalist | "previously unidentified" / "not noted in prior scholarship" |
| groundbreaking | Sensationalist | "original contribution" |
| proves beyond doubt | Overclaims | "supported by primary source evidence" |

## Preferred Language

| Concept | Preferred Phrasing |
|---|---|
| Strong claim well-supported by primary sources | "The primary source evidence supports the identification of..." |
| Interpretive synthesis | "The evidence is consistent with a reading in which..." |
| Candidate mechanism | "A candidate mechanism for this effect is..." |
| Provisional reconstruction | "The provisional reconstruction identifies..." |
| Speculative hypothesis | "As a working hypothesis, subject to further testing..." |
| Machine-mediated transfer (Shroud) | "machine-mediated transfer process" / "photothermal image-formation mechanism" |
| Interpretive hypothesis | "interpretive hypothesis pending replication" |
| Candidate density field | "candidate density field requiring further verification" |
| Restricted review | "submitted for specialist critique only" / "restricted specialist review" |
| Priority claim | "original contribution to scholarship not previously identified in published literature" |
| Not previously noted | "not identified in Reti (1974) or subsequent scholarship" |
| Evidence tier qualifier | "TYPE_A primary evidence supports..." / "the TYPE_C interpretive inference suggests..." |

## Figura Ultima Specific Constraints

The following language rules apply specifically to all Figura Ultima material and any public reference to the Turin Shroud thesis:

- **NEVER state:** "Leonardo da Vinci made / created / forged the Turin Shroud."
- **ALWAYS state:** "The Figura Ultima thesis proposes that the Turin Shroud image is consistent with a machine-mediated photothermal transfer process of a type documented in Leonardo's manuscripts. This is a testable model, not a proven fact."
- The thesis is described as: "a candidate framework under adversarial review."
- The 7.1 ratio is described as: "a mathematical relationship that, if confirmed by independent measurement, would constitute strong evidence for deliberate design."

## Review Statement (Required in All Reviewer Packets)

The following statement must appear in all packets distributed to reviewers:

> *"This material is submitted for specialist critique only. It does not carry institutional endorsement, DOI registration, or public validation. It does not imply academic acceptance. The findings are presented as testable models, interpretive hypotheses, and candidate frameworks for specialist critique. Peer review state: RESTRICTED SPECIALIST REVIEW — NOT PUBLIC VALIDATION."*
